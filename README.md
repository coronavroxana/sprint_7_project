# Aplicación Web de Análisis de Vehículos
Esta aplicación web permite analizar datos de anuncios de venta de coches mediante un histograma que muestra la distribución de odómetros y un gráfico de dispersión de la relación de los odómetros y el precio.
## Funcionalidades
- Visualización interactiva de datos de vehículos (histograma y gráfico de dispersión)
## Cómo usar
1. Selecciona el botón del gráfico que quieras visualizar.
